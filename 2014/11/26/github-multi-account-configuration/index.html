添加多个账户

上面操作时手工产生的id_rsa密钥文件放到username\.ssh\目录下，如果改成其它文件名，在连接时将出错，这就不能支持多个账号的ssh key。

（不过，我猜可能通过命令 ssh-add ~/.ssh/id_rsa 添加的 key 可能会自动产生多账号配置文件。有待验证！！！）
